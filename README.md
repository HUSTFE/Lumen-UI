# Lumen UI

> a UI Lib build with React

It's still not finished, you can see it on
[lumen on wulala](https://lumen.wula.la)

## Usage
It's a React Component UI Lib, you can install through npm or yarn

```bash
npm install --save lumen-ui
# or use yarn
yarn add lumen-ui
```
